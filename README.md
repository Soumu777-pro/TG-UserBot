## 💻 Features
- Multi-bot system: Startup Bot, Approval Bot, Control Bot
- MongoDB powered login & approval system
- Auto promo / broadcast commands
- Userbot session management
- 3-day auto disapprove system
- Ultra-fast performance optimized for Termux / VPS / Heroku

---

## 📜 Usage

1. Click **Deploy to Heroku** button above
2. Set environment variables:
   - `API_ID`, `API_HASH`
   - `STARTUP_BOT_TOKEN`, `APPROVAL_BOT_TOKEN`, `CONTROL_BOT_TOKEN`
   - `OWNER_ID`, `MONGO_URI`
3. Start the bot
4. Follow `/start` instructions in Telegram
